# Calc 🧮

사칙연산과 메모리 기능을 갖춘 단일 HTML 파일 계산기입니다. 별도 설치나 빌드 과정 없이 `index.html` 파일만 열면 바로 사용할 수 있습니다.

## 주요 기능

### 사칙연산
- 더하기(+), 빼기(−), 곱하기(×), 나누기(÷)
- 부호 전환(±), 백분율(%) 계산
- 소수점 입력 지원

### 메모리 기능
- **M+** : 현재 값을 메모리에 더하기
- **M−** : 현재 값을 메모리에서 빼기
- **MR** : 메모리에 저장된 값 불러오기
- **MC** : 메모리 초기화
- **메모리 내역 보기** : M+/M− 조작 시 처음 입력했던 시점부터의 모든 변경 내역과 누적 합계를 목록으로 확인 가능

### 키보드 입력 지원
마우스 없이 키보드만으로 모든 기능을 사용할 수 있습니다.

| 키 | 동작 |
|---|---|
| `0`–`9` | 숫자 입력 |
| `+`, `-`, `*`, `/` | 사칙연산자 |
| `.` | 소수점 |
| `%` | 백분율 |
| `Enter` / `=` | 계산 실행 |
| `Backspace` | 마지막 입력 삭제 |
| `Escape` | 전체 초기화 (C) |
| `Ctrl+M` | 메모리 더하기 (M+) |
| `Ctrl+Shift+M` | 메모리 빼기 (M−) |
| `Ctrl+R` | 메모리 불러오기 (MR) |
| `Alt+M` | 메모리 초기화 (MC) |

### 테마 전환
- 계산기 상단의 토글 버튼(☀️ 라이트 / 🌙 다크)으로 밝은 테마와 어두운 테마를 즉시 전환
- 선택한 테마는 `localStorage`에 저장되어 다음 방문 시에도 유지됨
- 별도 설정이 없으면 시스템의 다크모드 설정을 자동으로 따름

## 사용 방법
1. 이 저장소를 클론하거나 다운로드합니다.
2. `index.html` 파일을 웹 브라우저로 엽니다.

## 기술 스택
- HTML / CSS / Vanilla JavaScript (프레임워크 및 빌드 도구 없음)
